# Unidad Aritmética Basada en Pila

## 📋 Descripción
Unidad aritmética que utiliza una pila (stack) para realizar operaciones matemáticas básicas. Diseñada para estudiantes de informática como parte del curso de Estructuras de Datos.

## ✨ Características
- Operaciones soportadas: suma (+), resta (-), multiplicación (*), división (/)
- Manejo de números enteros y decimales
- Manejo de números negativos
- Validación de errores:
  - Elementos insuficientes en la pila
  - División entre cero
  - Entradas no válidas

## 🚀 Cómo ejecutar

### Requisitos
- Python 3.6 o superior

### Ejecutar el programa
```bash
python main.py
